# Curtain-wall aluminium case study

This directory contains the deterministic case-study implementation used in the journal paper.

## Case-study scope

- declared building-scale unit: **1 m² of installed curtain wall**
- representative aluminium content: **3.666 kg Al/m²**
- main analytical basis: **1 kg Al**
- decision context: **Canada**
- receiving systems:
  - contemporary (2025)
  - central prospective SSP2-M (2050)

## Route set

- landfill
- reuse
- conventional recycling via remelting
- MS-FSC
- aluminium hydrolysis to hydrogen, with conditional Al(OH)3 handling

## Folder roles

- `configs/` — case-study configuration files
- `scripts/` — case-study build and post-process scripts
- `inputs_processed/` — processed, shareable case-study input tables
- `results/` — machine-readable case-study result tables
- `figures/` — figure source tables and exported figure inputs
